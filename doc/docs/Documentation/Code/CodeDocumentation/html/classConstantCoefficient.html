Detailed Description

Constant-valued coefficient implementation.

This class represents a constant scalar-valued coefficient.

The class derives from FunctionCoefficient and overrides the function, gradient, and Hessian evaluators.

Hessian evaluator of the constant function.

The Hessian of a constant function is zero everywhere.

Remarks
The Hessian is stored in a flattened vector format: H(i,j) is stored at index i * n + j, where n is the number of variables.
