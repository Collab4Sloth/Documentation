Construct a new ThermalDiffusionFluxNLFormIntegrator object.

This constructor initializes the nonlinear form integrator. It forwards the provided previous solution fields, simulation parameters, auxiliary variables, and coefficients to the base DiffusionFlux nonlinear form integrator.
